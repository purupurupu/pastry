# Pastry 🥐

macOS向けのクリップボード履歴マネージャー。コピーした内容を自動で保存し、いつでも呼び出せます。

## 特徴

- **3種類のコンテンツ対応** - テキスト、画像、ファイルを自動検出・保存
- **メニューバー常駐** - Dockには表示されず、メニューバーから操作
- **グローバルショートカット** - どのアプリからでも `⌘+⌥+V` で即座に呼び出し
- **インクリメンタル検索** - 履歴をリアルタイムで絞り込み
- **キーボード操作** - 矢印キーで選択、Enterで貼り付け、`⌘+1〜9`でクイック選択
- **macOSネイティブUI** - バイブランシー効果によるモダンな見た目

## インストール

```bash
# 依存関係のインストール
npm install

# 開発モードで起動
npm start
```

## 配布用ビルド

```bash
# macOS向けにパッケージ化
npm run make
```

`out/make/` ディレクトリにdmgファイルが生成されます。

## 使い方

| 操作 | 説明 |
|------|------|
| `⌘+⌥+V` | Pastryウィンドウの表示/非表示 |
| `↑` / `↓` | 履歴の選択 |
| `Enter` | 選択したアイテムをクリップボードにコピー |
| `⌘+1〜9` | 上から1〜9番目のアイテムをコピー |
| `Backspace` | 選択したアイテムを削除（検索欄が空の時） |
| `Escape` | 検索をクリア |

## 設定

ウィンドウ右上の歯車アイコンから設定画面を開けます。

- **履歴上限** - 保存するアイテム数（10〜1000）
- **グローバルショートカット** - 呼び出しキーの変更
- **履歴クリア** - すべての履歴を削除
